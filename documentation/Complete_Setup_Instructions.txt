ğŸ› COMMON ERRORS & SOLUTIONS
================================================================================

ERROR: "ModuleNotFoundError: No module named 'mysql'"
SOLUTION: Install the package
    pip install mysql-connector-python

ERROR: "Access denied for user 'root'@'localhost'"
SOLUTION: Wrong password in common.py (line 23)
    - Check your MySQL password
    - Update self.password in common.py

ERROR: "Unknown database 'zoo_management'"
SOLUTION: Database not created
    - Run database_setup.sql in MySQL Workbench first

ERROR: "Can't connect to MySQL server"
SOLUTION: MySQL is not running
    - Open MySQL Workbench to start the server
    - Or start MySQL service from Services (Windows)

ERROR: "No module named 'login_integrated'"
SOLUTION: Missing file
    - Make sure all 10 Python files are in the same folder
    - Check file names match exactly (case-sensitive!)

ERROR: Last line in common.py causes error
SOLUTION: Remove these lines from bottom of common.py:
    if __name__ == "__main__":
        root = tk.Tk()
        app = ZooManagementSystem(root)
        root.mainloop()
