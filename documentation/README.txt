PROJECT DESCRIPTION:
This is a Zoo Management System built with Python Tkinter and MySQL.
It allows administrators to manage animals, staff, feeding schedules,
health records, and enclosures.

FOLDER STRUCTURE:
- src/          : All Python source code files
- database/     : SQL database setup script
- documentation/: Project documentation and reports
- screenshots/  : Screenshots of working system

SYSTEM REQUIREMENTS:
- Python 3.6 or higher
- MySQL Server 8.0 or higher
- mysql-connector-python package

HOW TO RUN:
1. Setup database using database/database_setup.sql
2. Update MySQL password in src/common.py (line 23)
3. Run: python src/main.py

DEFAULT LOGIN:
Username: admin
Password: admin123

FEATURES:
✓ User authentication (Admin/Staff)
✓ Animal records management (CRUD operations)
✓ Feeding schedule management
✓ Health records tracking
✓ Enclosure management
✓ Staff management
✓ Task assignment system
✓ Dashboard with statistics

TECHNOLOGIES USED:
- Python 3.x
- Tkinter (GUI)
- MySQL (Database)
- mysql-connector-python (Database connector)
